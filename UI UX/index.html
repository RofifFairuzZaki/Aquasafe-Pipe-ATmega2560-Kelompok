Tentang Proyek AquaSafe Pipe

Sistem ini dirancang menggunakan pendekatan sistem tertanam (embedded system) berbasis arsitektur bare-metal AVR mikrokonter ATmega2560. Parameter ambang batas risiko ditentukan secara langsung lewat register internal untuk efisiensi memori tingkat tinggi.

Algoritma Klasifikasi Risiko

  • Kadar pH < 6.50 menambah beban skor akumulasi +40 poin.
  • Padatan Terlarut TDS > 500 ppm menambah beban skor akumulasi +35 poin.
  • Tingkat Kekeruhan Turbidity > 50% menambah beban skor akumulasi +25 poin.

Target MCU: ATmega2560 Bare-metal AVR Architecture

Serial Core: UART0 Channel @9600 BAUD
